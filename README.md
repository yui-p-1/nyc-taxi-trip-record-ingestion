# NYC Taxi Data Ingestion Pipeline

This project builds an automated data ingestion pipeline for the NYC Taxi dataset.

## Architecture
Cloud Scheduler
→ Cloud Run (Python)
→ Google Cloud Storage
→ BigQuery External Tables
→ Dataform Transformation
→ Tableau Dashboard

## Technologies
- Google Cloud Run
- Google Cloud Storage
- BigQuery
- Dataform
- Tableau

## Data Source
NYC Taxi & Limousine Commission Trip Record Data
